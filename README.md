# Job Application Tracker (Vanilla JavaScript)

A clean, state-driven job application tracker built with vanilla JavaScript. This project focuses on core frontend fundamentals such as state management, DOM rendering, and data persistence without relying on frameworks.

Users can add, edit, and delete job applications, track their status (Applied, Interview, Rejected), and filter results dynamically. All data is stored locally using `localStorage`, allowing the app to persist between sessions.

## Features

- Add, edit, and delete job applications
- Status tracking with visual badges
- Filter applications by status
- Persistent data using `localStorage`
- Event delegation for scalable interactions
- Fully state-driven rendering (State → Render → DOM)

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript (no frameworks)
